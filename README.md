# Pang AI

**Pang AI** is an AI extension for PenguinMod. It provides all the blocks you need for your AI projects, designed to be accessible for everyone.

I am excited to help people learn how to use AI components at an early age. You can use this however you like, with no prompt limits.

## All Blocks

| Block Name | Type | Description |
| :--- | :--- | :--- |
| **AI ready?** | Boolean | Returns true if the AI server is online. |
| **when AI connects** | Hat | Starts a script when the AI is ready. |
| **when AI disconnects** | Hat | Starts a script if the AI goes offline. |
| **set system log to [ ]** | Command | Changes the AI's personality/identity. |
| **set model to ( )** | Command | Changes the AI's "brain" (OpenAI, etc). |
| **ask prompt [ ]** | Reporter | Gets a text answer from the AI. |
| **get url of image [ ]** | Reporter | Gets a link to an AI-generated image. |
| **set temperature to ( )** | Command | Changes how creative or random the AI is. |
| **set seed to ( )** | Command | Sets a fixed number for the AI's logic consistency. |
| **temperature** | Reporter | Shows the current creativity setting. |
| **seed** | Reporter | Shows the current seed number. |
